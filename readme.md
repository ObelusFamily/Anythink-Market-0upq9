## Run Locally

Clone the project

```bash
  git clone https://github.com/GautamSharma05/SharmaStoreNextWebsite.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  install Docker
```

Start the server

```bash
  docker-compose up
```
